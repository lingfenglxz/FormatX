# 右侧结构树标签设计

## 目标

将 JSON 结构树的容器节点标签改为图一的紧凑样式，不改变现有编辑、搜索或展开逻辑。

## 展示规则

- 根对象显示为 `object {N}`，其中 `N` 为根对象属性数。
- 对象子节点显示为 `key {N}`；数组子节点显示为 `key [N]`。
- 无论节点展开还是收起，数量都紧跟在节点名右侧。
- 不显示独立的开闭括号、`...`，或“项/键”等文字说明。
- 基础值、节点展开按钮、缩进及双击编辑保持原样。

## 范围

仅修改 `src/App.tsx` 中现有的 `JsonTree` 渲染和 `src/App.test.tsx` 中与该标签规则对应的测试；不引入依赖或替换树组件。

## 验证

用一个含对象与数组的 JSON 测试根节点、收起节点和展开节点均输出紧凑数量标签，并运行完整前端测试及生产构建。
